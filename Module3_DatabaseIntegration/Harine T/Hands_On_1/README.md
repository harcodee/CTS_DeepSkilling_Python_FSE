# Module 3 – Database Integration (Hands-On 1)

## Student Course Registration System

This project implements the **Student Course Registration System** database using **MySQL** as part of the Digital Nurture 5.0 Python Full Stack Engineer Track.

## Contents

* Database creation (`college_db`)
* Creation of five relational tables:

  * `departments`
  * `students`
  * `courses`
  * `enrollments`
  * `professors`
* Primary Key, Foreign Key, UNIQUE, and NOT NULL constraints
* Sample data insertion
* Verification of 1NF, 2NF, and 3NF normalization
* Schema modification using `ALTER TABLE`
* Addition and removal of columns
* CHECK constraint implementation
* Column renaming

## Technologies Used

* MySQL 8.x
* SQL (DDL & DML)

## Files

* `hands_on_1.sql` – Complete SQL solution for Hands-On 1
* `README.md` – Project documentation

## How to Run

1. Open MySQL Workbench.
2. Execute the `hands_on_1.sql` file.
3. The script will:

   * Create the `college_db` database.
   * Create all required tables.
   * Insert sample data.
   * Verify normalization through SQL comments.
   * Perform ALTER TABLE operations.

## Learning Outcomes

* Database schema design
* Referential integrity using foreign keys
* SQL DDL operations
* Database normalization (1NF, 2NF, 3NF)
* Schema modification using ALTER TABLE
